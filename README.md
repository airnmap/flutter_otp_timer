# flutter_otp_timer

Get easy otp timer in your app

# Usage

Import `package:flutter_otp_timer/flutter_otp_timer`

Example:

```dart
import 'package:flutter_otp_timer/flutter_otp_timer';


```

You will find links to the API docs on the [pub page](https://pub.dev/packages/otp_count_down).
